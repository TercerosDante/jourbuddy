# jourbuddy.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)
- yarn

### Installation

```bash
# Install dependencies for all packages
yarn
```

### Development

To run both the frontend and backend in development mode:

```bash
yarn dev
```

This will start:
- Frontend development server (Vite)
- Backend development server (Nodemon)
